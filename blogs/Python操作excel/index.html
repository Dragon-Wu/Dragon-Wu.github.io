Pythonж“ҚдҪңexcel

еҸ‘иЎЁдәҺ 2019-11-04 | еҲҶзұ»дәҺ CS | | йҳ…иҜ»ж•° ж¬Ў
еӯ—ж•°з»ҹи®Ў: 1.1k | йҳ…иҜ»ж—¶й•ҝ ≈ 4

жңҖиҝ‘иҜ»еҸ–ж•°жҚ®е’Ңеё®еҠ©еёҲе…„еёҲе§җеҒҡдёҖдәӣз®ҖеҚ•зҡ„з”ҹдҝЎдҝЎжҒҜеҲҶжһҗйғҪйңҖиҰҒз”ЁеҲ°excelи§Јжһҗж•°жҚ®пјҢж•…еҜ№pythonж“ҚдҪңexcelзҡ„е·Ҙе…·еҒҡдёӘж•ҙзҗҶ

дҪҝз”Ёе“ӘдёӘеә“пјҹ

е…¶е®һдёҖжҗңе°ұеҸ‘зҺ°пјҢPythonж“ҚдҪңexcelзҡ„еҗ„дёӘж•ҷеӯҰеҚҡж–Үиӣ®еӨҡзҡ„пјҢдё»иҰҒе°ұжҳҜxlrdгҖҒxlwtе’ҢopenpyxlдёӨзұ»пјҢжҺҘдёӢжқҘе°ұдё»иҰҒи®Іи®ІеҰӮдҪ•иҜ„дј°дәҢиҖ…

еҗ„дёӘеә“зҡ„д»Ӣз»Қ

xlrd

и·Ёе№іеҸ°зҡ„pythonжЁЎеқ—пјҢз„¶еҗҺжңүзү№ж„ҸжҸҗеҲ°д»–more like a native python coding styleпјҢдё”ж”ҜжҢҒxlsе’ҢxlsxпјҢдҪҶжҳҜеҸӘжҳҜиҜ»еҸ–

( It is a Python Module for dealing with Excel files that can work on Windows, Linux as well as Mac platforms. It has more like a native python coding style and very helpful in accessing all types of excel files if they are xls or even xlsx. However it is used only for reading purposes. )

xlwt

з®—жҳҜxlrdзҡ„й…ҚеҘ—пјҢд»…ж”ҜжҢҒExcel95-2003зҡ„зј–иҫ‘пјҢжүҖд»ҘxlwtдёҚиғҪеҶҷе…Ҙи¶…иҝҮ65535иЎҢгҖҒ256еҲ—зҡ„ж•°жҚ®

д№ҹжҸҗеҲ°дәҶд»–зҡ„йЈҺж јжҳҜе…Ёpythonзҡ„пјҢpythonerеҸҜиғҪдјҡжӣҙе–ңж¬ўд»–зҡ„еҺҹз”ҹи§Ұж„ҹ

( xlwt library allows developers to create spreadsheets compatible to Excel 95-2003. Hence we can say that it can be used for older files. Module is in pure Python form and exists as independent entity. Python programmers like to code in it as it has a native touch and taste. )

xlsxwriter

дёҠйқўзҡ„xlwtеҸӘиғҪж”ҜжҢҒеҲ°xlsпјҢдҪҶжҳҜзҺ°еңЁеӨ§еӨҡжҳҜxlsxпјҢиҝҷдёӘеә“е°ұжҳҜж–°еўһз”ЁдәҺxlsxзҡ„дҪҝз”Ё

жғҠй—»еҷ©иҖ—пјҡе®ғеҸӘиғҪеҶҷе…Ҙж–°ж–Үд»¶пјҒпјҒпјҒдёҚеҸҜдҝ®ж”№ж–ҮжЎЈ

( It is an alternative Python module for Writing data in Excel file. It specializes in formatting of data and it can also be used for charts in Excel 2010 xlsx formats. It is a wonderful package with many qualites, which if truly harnessed can do magic in terms of data writing and formatting. )

openpyxl

ж–°зҡ„pythonж“ҚдҪңexcelеә“пјҢдё»иҰҒз”ұ Eric Gazoni and Charlie ClarkејҖеҸ‘пјҢејҖжәҗдё”е…Қиҙ№пјҢж”ҜжҢҒ Excel xlsx/xlsm/xltm/xltx files without using the Excel softwareпјҢиҖҢдё”д»–жҳҜж”ҜжҢҒиҜ»еҶҷзҡ„пјҢд№ҹе°ұжҳҜдёҚз”ЁеғҸдёҠиҝ°зҡ„еә“еҸӘиғҪиҜ»жҲ–иҖ…еҶҷпјҢд»Ӣз»Қе°ұдё»иҰҒи®ІдәҶе•Ҙexcelж“ҚдҪңеҫҲеӨҚжқӮпјҢ it may take hours, and if the files are hundreds or thousands it can take days or sometimes weeks дҪҶжҳҜдҪ иҰҒжҳҜз”ЁдәҶжҲ‘д»¬зҡ„еә“пјҢ you just write a simple program usually spending a few minutes and your hectic boring work is done by computer in no time пјҢзүӣйҖјеҗ§гҖӮгҖӮгҖӮдәӢе®һдёҠжҲ‘и§үеҫ—еҫҲеӨҡж—¶еҖҷпјҢexcelзҡ„еҫҲеӨҡе·Ҙе…·е’ҢеҮҪж•°еҫҲж–№дҫҝзҡ„пјҢдё»иҰҒжҳҜдёҖж–№йқўзңӢдҪ е“Әж–№йқўжҜ”иҫғеҺүе®іпјҢеҸҰдёҖж–№йқўжҳҜзңӢдҪ йңҖдёҚйңҖиҰҒе…¶д»–еӨ„зҗҶе’ҢжҺҘеҸЈ

жңүдёҖдёӘдёҚж–№дҫҝзҡ„пјҢд»–дёҚж”ҜжҢҒж—©жңҹзҡ„xls

е…¶е®һеҫҲеҘҪеҘҮиҝҷдҝ©зҪ‘з«ҷе’ӢжқҘзҡ„пјҢз„¶еҗҺеүҚиҖ…зҪ‘з«ҷж„ҹи§үжҳҜopenpyxlж•ҙзҡ„пјҢеӣ дёәеҗ№дәҶеҫҲеӨҡе®ғпјҢе№¶дё”иҝҳжңүе…¶ж•ҷеӯҰиҜҫзЁӢзҡ„й“ҫжҺҘпјҢ10еҲҖгҖӮгҖӮжҲ‘з”ҡиҮіжІЎжүҫеҲ°д»–зҡ„referenceй“ҫжҺҘпјҢиҝҷдёӘиҝҳжҢәзғҰзҡ„ = =

жқғиЎЎ

е…је®№жҖ§

йҰ–е…ҲopenpyxlпјҢдёҖзӣҙиў«еҗ„з§Қе®үеҲ©пјҢжҲ‘и§үеҫ—еҘҪеӨ„жҳҜжңүзҡ„пјҢжҜ•з«ҹжҜ”иҫғж–°пјҢиҖҢдё”иҝӣиЎҢдәҶж•ҙеҗҲпјҢиҜ»еҶҷеҗҢдёҖдёӘеә“пјҢеҗҢж—¶еҜ№еҗ„дёӘзүҲжң¬зҡ„excelж–Үд»¶ж јејҸйғҪе…је®№пјҢиҝҷдёҖзӮ№дёҠиҝҳжҳҜжңүиҫғеӨ§дјҳеҠҝзҡ„

жү©еұ•жҖ§

openpyxlзү№ж„ҸжҸҗеҲ° openpyxl is the default reader for Python Pandas пјҢиҖҢдё”зңӢreferenceйҮҢжңүеҫҲеӨҡе…ідәҺpandaгҖҒnumpyзҡ„жҺҘеҸЈпјҢжҜ”еҰӮи®ІworksheetзӣҙжҺҘиҪ¬жҲҗdataframeзӯүпјҢиҝҷдёҖзӮ№дёҠпјҢopenpyxlзҡ„дјҳзӮ№иҝҳжҢәжҳҺжҳҫзҡ„гҖӮеҪ“з„¶пјҢжҲ‘иҝҳжІЎз”ЁиҝҮxlrdпјҢдёҚзҹҘйҒ“д»–д»¬зҡ„ж”ҜжҢҒжғ…еҶө

жҖ§иғҪ

иҝҳжңүжңҖе…ій”®зҡ„жҖ§иғҪй—®йўҳпјҢиҝҷдёҖзӮ№openpyxlиҝҳжҳҜжҢәиҖҒе®һзҡ„пјҢиҮӘе·ұеқҰзҷҪдәҶиҜҙжҲ‘д»¬иҝҷдёӘеә“пјҢеңЁжқғиЎЎжҖ§иғҪе’ҢеҠҹиғҪдёҠпјҢйҖүжӢ©дәҶеҠҹиғҪпјҲиҮҙж•¬pythonпјҹпјҢжүҖд»ҘжҖ§иғҪдёҠзЎ®е®һжңүж¬ зјәпјҢе…·дҪ“еҸҜд»ҘзңӢhttps://openpyxl.readthedocs.io/en/stable/performance.htmlпјҢе°ұжҸҗеҲ°дәҶopenpyxlе’Ңxlrd/xlwtзҡ„жҜ”иҫғпјҢжҖ»дҪ“жқҘиҜҙеүҚиҖ…иҝҳжҳҜж…ўдәӣпјҢдҪҶжҳҜжіЁж„ҸдёҖзӮ№пјҡеҸҜд»ҘеҶҚиҜ»еҶҷж—¶жіЁжҳҺеҸӘиҜ»жҲ–иҖ…еҸӘеҶҷпјҢиғҪеҝ«дәӣпјҢдҪҶжҳҜopenpyxlеңЁеҸӘеҶҷзҡ„ж—¶еҖҷпјҢеҸӘиғҪappendпјҢдёҚиғҪжҸ’е…ҘгҖӮ
